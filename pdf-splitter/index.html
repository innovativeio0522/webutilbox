What Is a PDF Splitter?

A PDF splitter is a tool that takes a multi-page PDF document and separates it into smaller files. You can split every page into its own individual PDF, extract a specific range of pages (for example, pages 3 through 7), or cherry-pick individual pages by number (such as pages 1, 4, and 9).

This is the reverse operation of a PDF merger. Where a merger combines files, a splitter divides them. Both operations are essential for anyone who regularly works with PDF documents in professional, academic, or personal contexts.

This tool runs entirely in your browser using the pdf-lib JavaScript library. It reads your PDF locally, processes the page extraction on your device, and delivers the output files as instant downloads — all without any server involvement.

Tips and Best Practices

When using the custom page input, you can combine individual page numbers and ranges in a single entry. For example, entering "1, 3, 5-8, 12" will extract pages 1, 3, 5, 6, 7, 8, and 12 into a single output PDF in that order. This gives you precise control over exactly which content ends up in your output file.

If you need to split a very large PDF into many individual pages, be aware that your browser will trigger multiple downloads — one per page. Some browsers may ask for permission to download multiple files at once. Allow this when prompted to ensure all pages are saved.

Password-protected PDFs cannot be split until the password is removed. If your PDF is encrypted, use a PDF unlock tool first, then return here to split it. Also note that page numbers in this tool are 1-indexed, meaning the first page is page 1, not page 0.
